# 1stKontact

> Official website for illustrator こんたくん (Kontakun).

[🔗 Live Site](https://1st-kontact.vercel.app)

## ✨ Features

- 🖼 Gallery with illustration showcase
- 🎨 Works portfolio
- 📇 Profile & contact pages

## 🛠 Tech Stack

- Next.js (App Router) + React + TypeScript
- Tailwind CSS

## 🚀 Development

```bash
pnpm install
pnpm dev
```
